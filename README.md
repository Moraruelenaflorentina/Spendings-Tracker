#  SpendingsTracker

A simple and straightforward Windows desktop application for tracking your income and expenses — so you always know where your money is going.

>  **Note:** This project is currently an early prototype. Features and the UI are still being developed and may change.

---

##  Description

**SpendingsTracker** is a Windows Forms Application (WFA) built with C# and .NET. It allows anyone to easily log their income and expenses, giving them a clearer picture of their personal finances without the complexity of full-scale accounting software.

Whether you want to keep tabs on your monthly budget or just see how much you've been spending, SpendingsTracker keeps it simple and accessible.

---

##  Features (Current)

- Log income entries
- Log expense entries
- View a summary of your financial activity
- Lightweight desktop app — no internet required

---

##  How to Use

### 1. Launch the App
Open `SpendingsTracker.exe` (or run the project from Visual Studio).

### 2. Add Category
- Navigate to the **Category** section
- Enter the name, tipe, and status
- Click **Add** to save the entry
- 
### 2. Add Income
- Navigate to the **Income** section
- Enter the amount, date, and a short description
- Click **Add** to save the entry

### 3. Add an Expense
- Navigate to the **Expenses** section
- Enter the amount, date, category, and description
- Click **Add** to save the entry

### 4. View Your Summary
- Go to the **Dasboard** section to see your total income, total expenses, and remaining balance

##  Getting Started

### Prerequisites
- Windows OS
- [.NET Desktop Runtime](https://dotnet.microsoft.com/en-us/download) (version used in this project)
- Visual Studio (if running from source)

### Run from Source
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/SpendingsTracker.git
   ```
2. Open `SpendingsTracker.sln` in Visual Studio
3. Press **F5** to build and run
